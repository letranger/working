2. 跨域素養

2.1. 資訊科技

  1. 運 a-V-2 能使用多元的觀點評論資訊科技相關議題。
  2. 運 a-V-3 能探索新興的資訊科技。
  3. 運 c-V-1 能使用資訊科技增進團隊合作效率。
  4. 運 p-V-1 能整合資訊科技進行有效的溝通表達。
  5. 運 t-V-2 能使用程式設計實現運算思維的解題方法。
  6. 運 t-V-3 能應用運算思維評估解題方法的優劣。
  7. ※運 a-V-5 能主動探索資訊科技新知。
  8. ※設 c-V-3 能具備溝通協調、組織工作團隊的能力。
  9. ※設 k-V-2 能了解科技產業現況及新興科技發展趨勢。
  10. ※運 r-V-3能利用程式語言表達運算程序

2.2. 數學科

  1. f-V-7: 理解導函數的意義，熟練其操作，並能用以解決問題。
  2. f-V-8: 認識微分與積分互為逆運算，理解微積分基本定理的意義，並能用以推論。

3. 學習目標

  1. 資料預處理與統計分析技能：學生能理解並應用資料清理、正規化及基本統計分析方法，為後續AI模型訓練做好準備。
  2. AI模型的運作原理：學生理解分類與迴歸模型的原理，並能將這些技術應用於預測段考成績與學測級分。
  3. 實作AI模型：學生能實作包括KNN、決策樹、感知器、神經網絡、YOLO、LSTM、RNN等多種AI模型，並應用這些模型進行實際預測。
  4. 優化與評估模型：學生學會如何通過超參數調整來優化模型效能，並能評估不同模型的預測準確度，找出最佳解法。
  5. 跨學科知識整合：學生能結合數學知識（如梯度下降法）與AI技術，深入理解AI模型的數學基礎。
  6. 專題實作與發表：學生在期末專題中應用所學知識進行實作，並能展示專題成果，提升協作與表達能力。

4. 教學大綱

4.1. 第1週: Introduction to AI

  • 本學期課程內容規畫簡介
  • 了解AI的運作原理與建構流程、認識不同AI類型以及其應用領域。
  • 介紹AI模型的基本概念，講解分類與迴歸的差異(段考成績預測為迴歸、學測級分預測為分類)，並簡介分類技術。
  • 理解資料正確性對於預測的重要性，進而了解資料預處理的必要性。
  • 卜聖卦活動1: 讓學生自行預測自己本學期(2下)第一次段考的各科成績，並讓學生列出自己做出預測的依據

4.2. 第2週: Statistical Analysis and Regression Prediction

  • [內聘講師/數學科]介紹基本統計分析方法，並進行數據的初步分析，然後引入數學迴歸分析，讓學生進行第一次段考的數據預測。
  • 卜聖卦活動2: 學生根據歷屆段考成績進行數學迴歸分析計算，以自己的一年級與二上成績預測二下第一次段考的成績，並與 卜聖卦活動1的結果 比較，討論預測依據的差異。

4.3. 第3週: Data Preprocessing and Cleaning

  • 介紹資料清理與預處理的基本概念，包括處理缺失值、數據標準化、正規化，並收集歷屆段考成績作為參考資料。
  • 學生收集學校提供的歷屆段考成績數據，並進行資料預處理實作，為後續模型建立做準備。

4.4. 第4週: Introduction to Supervised Learning

  • 介紹AI實作迴歸的模型及運作原理
  • 卜聖卦活動3: 以AI迴歸模型來預測二下第一次段考成績，並與前兩次的預測結果比較，討論預測依據的差異。

4.5. 第5週: Introduction to Supervised Learning

  • 介紹AI實作分類的模型及運作原理
  • 介紹K最近鄰（KNN）分類模型的原理，並讓學生實作KNN模型來預測學測成績級分(依據一年級全學年及二上的成績)。
  • 介紹決策樹模型的原理及其應用，學生學習如何利用決策樹進行學測成績級分預測。

4.6. 第6週: Introduction to Unsupervised Learning

  • 卜聖卦活動4: 依據學生二下第一次段考實際成績評估前三次(卜聖卦活動1, 卜聖卦活動2, 卜聖卦活動3)預測的準確性
  • 了解非監督式學習的概念與運作原理。
  • 能實做簡單的非監督式學習模型。
  • 以K-Means技術實作影像壓縮

4.7. 第7週: Perceptron: The Foundation of Neural Networks

  • 介紹神經網絡的基本概念與運作原理，了解如何將其應用到各項領域。
  • 能實作神經網路中的感知器。

4.8. 第8週: Introduction to Deep Learning

  • 能理解神經網路與深度學習的關係
  • 活動：神經網絡實作競賽: 學生實作神經網絡模型，並比賽誰的模型能最準確地預測下一次段考的成績。

4.9. 第9週: Introduction to Deep Learning and CNN

  • 介紹卷積神經網絡（CNN）的概念，並了解其在圖像分類中的應用，為學生理解模型架構打下基礎。
  • 了解CNN的運作原理。
  • 展示CNN在不同領域的應用，學生討論CNN是否適合段考分數與學測預測。

4.10. 第10週: Mathematics Special Topic: Mathematical Explanation of Gradient Descent

  • [內聘講師/數學科] 介紹梯度下降法的數學基礎，講解微分與導數的概念，並以數學公式解釋梯度下降法的原理。

4.11. 第11週: You Only Look Once

  • 介紹YOLO (You Only Look Once) 演算法的基本概念及其在即時物體偵測中的應用。
  • 舉例說明YOLO在即時影像、監控系統、交通管理等領域的實際應用。
  • 學生實作YOLO模型進行即時物體偵測。

4.12. 第12週: LSTM與RNN簡介及實作

  • 介紹長短期記憶（LSTM）和遞歸神經網絡（RNN）的原理，理解其在時間序列數據中的應用
  • 學生實作LSTM/RNN模型來預測學測成績。
  • 卜聖卦活動5: 依據學生一年級全學年、二上以及二下第一次段考成績，以LSTM/RNN預測二下第二次段考成績

4.13. 第13週: AI Prediction: Unveiling the Sacred Exam Outcome

  • 學生進行模型的參數調整與優化，學習如何提升分類模型的預測精度。
  • 參數調整競賽: 學生嘗試調整模型參數，提升模型預測效果，並比較各組模型的預測效果。
  • 卜聖卦活動6: 學生分組(跨校異質分組)，以學校提供之歷屆學生段考與學測成績做為訓練資料，建立各組的預測模型。

4.14. 第14週: One Model Isn’t Enough? Let’s Use More!

  • 卜聖卦活動6結果實測: 教師以額外的測試資料對各組模型進行效能測試，請模型效能表現優異各組發表模型設計原理。
  • 介紹隨機森林模型的概念，並進行隨機森林的實作與模型優化。
  • 卜聖卦最終回: 各組以隨機森林實作出最後版本的學測級分預測模型

4.15. 第15週: 期末專題準備

  • 卜聖卦最終回結果揭曉: 教師以額外的測試資料對各組最終模型進行效能測試。
  • 學生分組進行期末專題的準備，探討專題主題與所需資料與應用模型。

4.16. 第16週: 期末專題實作

  • 學生進行期末專題實作，完成分類模型的構建，並準備專題報告。

4.17. 第17週: 期末專題實作

  • 學生進行期末專題實作，完成分類模型的構建，並準備專題報告。

4.18. 第18週: 期末專題成果發表

  • 學生展示專題成果，各組發表專題實作的成果及學習心得，發表過程進行線上直播，邀請學生家長及校內同學上線觀看。

5. 學習評量

5.1. 卜聖卦活動1：自我預測 (Self-Assessment Prediction)

  • 預測依據的合理性 (50%)：學生是否能清楚列出自己預測段考成績時的依據（如過去成績、學習習慣、學習時間等），並給出合理的解釋。
  • 預測準確度 (50%)：比較學生預測成績與實際段考成績，根據預測準確度進行評分。

5.2. 卜聖卦活動2：數學迴歸分析預測 (Regression Prediction)

  • 數據處理能力 (30%)：學生是否正確進行數據的清理與預處理，並運用適當的迴歸方法進行預測。
  • 模型選擇與解釋 (30%)：學生是否能清楚解釋為什麼選擇某種迴歸模型，並合理說明預測過程中的假設與限制。
  • 預測準確度 (40%)：根據迴歸分析結果與實際段考成績的對比，計算預測的準確性。

5.3. 卜聖卦活動3：AI迴歸模型預測 (AI Regression Model Prediction)

  • 模型實作能力 (30%)：學生是否能成功實作迴歸模型，並正確應用數據進行訓練和預測。
  • 模型調參與優化 (30%)：學生是否對模型參數進行合理調整，並嘗試優化預測效果。
  • 預測準確度 (40%)：比較AI模型的預測結果與實際成績，計算模型的準確度，並分析準確度的高低。

5.4. 卜聖卦活動4：KNN與決策樹模型的學測預測 (KNN and Decision Tree Prediction for Entrance Exam Scores)

  • 模型實作能力 (30%)：學生是否能成功實作KNN與決策樹模型，並正確應用模型進行學測成績級分的預測。
  • 模型比較與選擇 (30%)：學生是否能對比KNN與決策樹模型的優缺點，並能合理選擇哪個模型更適合特定數據集。
  • 預測準確度 (40%)：根據預測結果與實際歷屆學測成績的對比，計算預測的準確度。

5.5. 卜聖卦活動5：LSTM/RNN模型的段考預測 (LSTM/RNN Model Prediction for Exams)

  • 模型實作與應用 (30%)：學生能否成功實作LSTM/RNN模型，並正確使用時間序列數據進行預測。
  • 數據處理與預測 (30%)：學生能否正確處理時間序列數據，並進行準確的預測。
  • 預測準確度 (40%)：LSTM/RNN模型的預測結果與實際段考成績的比較，並計算準確性。

5.6. 卜聖卦活動6：隨機森林模型的學測預測 (Random Forest Prediction for Entrance Exam Scores)

  • 模型實作與應用 (30%)：學生是否能成功實作隨機森林模型，並正確應用來預測學測成績級分。
  • 模型優化與調參 (30%)：學生是否能有效地進行參數調整與模型優化，提升預測的準確度。
  • 預測準確度 (40%)：根據隨機森林模型的預測結果與實際成績進行比較，評估預測準確性。

5.7. 期末專題實作與發表 (Final Project Implementation and Presentation)

  • 專題模型構建與應用 (40%)：學生能否成功完成專題模型的構建，並應用所學技術進行有效的預測。專題應涵蓋數據預處理、模型實作、參數調整和結果分析等部分。
  • 專題創新性 (30%)：專題中有無創新性的應用，學生是否嘗試探索新技術或新的應用場景，並結合不同技術（如YOLO、LSTM等）進行預測。
  • 專題報告與成果展示 (30%)：學生的專題報告是否結構清晰、表達充分，並在成果展示中能有效解釋專題內容與核心技術。
